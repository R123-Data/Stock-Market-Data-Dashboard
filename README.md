Title: Stock-Market-Data-Dashboard

Description: This project presents an interactive Stock Data Analysis Dashboard developed to analyze portfolio performance, investment distribution, stock returns, and trading activity across multiple companies and sectors. The dashboard enables users to track key financial metrics and gain actionable insights through dynamic visualizations.

Technologies used:
Power BI - Main Data Visualization Platform used for report creation. 
Power Query - Data transformation and cleaning layer for reshaping and preparing the data. 
DAX (Data Analysis Expressions) - Used for calcuated measures,dynamic visuals, and conditional logic. 
Data Modeling - Relationships established among tables to enable cross-filteringand aggregation. 
Interactive Visualizations 
KPI Cards & Slicers

Portfolio Performance KPIs :
Total Market Capitalization: 417M
Portfolio Value: 944K
Total Investment: 834.82K
Total Profit & Loss: 109.29K
Return Percentage: 13%
Average Daily Volume: 25.41K
Total Traded Value: 135.13M

Sector Analysis:
Sector-wise Profit & Loss comparison.
Investment distribution across sectors.
Identification of top-performing sectors such as Energy and IT.
Company-Level Insights:
P&L analysis by company.
Return percentage comparison among companies.
Performance tracking of major stocks including Reliance, TCS, Infosys, Apple, Microsoft, Amazon, and others.
Exchange Analysis:
Investment comparison between NSE and NASDAQ.
Exchange-wise profitability analysis.
Portfolio Allocation:
Treemap visualization showing stock allocation by value.
Easy identification of high-weightage holdings within the portfolio.
Trading Trend Analysis:
Monthly traded value trend visualization.
Helps monitor market activity and trading behavior over time.

Insights:
Energy and IT sectors contributed the highest profits.
Reliance Industries represented the largest portfolio allocation.
NSE generated significantly higher investment and P&L compared to NASDAQ.
Portfolio delivered a positive return with strong performance from selected stocks.

Through this project, I learned how to transform stock market data into meaningful insights that help investors evaluate portfolio performance, monitor risk, and make data-driven investment decisions.
